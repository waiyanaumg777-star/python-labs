# Week 5 Documentation

## 1.1 Problem Statement
Create a Python program to calculate a customer's cafe bill automatically.

## 1.2 Inputs
- Customer name
- Coffee quantity
- Tea quantity
- Sandwich quantity

## 1.3 Outputs
- Receipt
- Total bill

## 1.4 Process Flow
1. Enter customer name.
2. Enter quantities.
3. Calculate total.
4. Print receipt.

## 1.5 Constraints
- Quantities must be positive.
- Coffee = RM8.50
- Tea = RM6.00
- Sandwich = RM12.00

## 2. Problem Decomposition
- Input customer information.
- Calculate total.
- Display receipt.

## 3. Pseudocode

START

Input customer name

Input coffee quantity

Input tea quantity

Input sandwich quantity

Calculate total

Print receipt

END
